GRUNDLÆGGENDE ANIMATION

I grundlæggende animation blev vi introduceret til javascript i forbindelse med at skulle udvikle eget animationsspil, og få færdigheder indenfor css animationer. Temaet var opdelt i flere faser. I første fase arbejdede jeg med koncept og idé til mit spil. Herefter udviklede jeg de grafiske elementer, som skulle indgå i mit spil. Senere kodede jeg en html og css side, hvor jeg placerede alle spil og UI elements. Jeg fik til dels animeret nogle af spil-elementerne.
